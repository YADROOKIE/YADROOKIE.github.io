git多设备同步代码

发表于 2020-01-10 | 分类于 git

放假回家，喜欢用台式机敲代码，但是有时也要笔记本来。
代码托管到GitHub通过GitHub同步代码。让我随时随地码起来

一

在台式机上安装git 并配置好ssh（笔记本上以及有git）

二

将代码在github中建好仓库并托管上去。

三

在台式机上git clone 仓库

四

当在一台电脑编码完并部署到GitHub上后。
在另一台电脑直接git pull master origin 就会自动完成拉取合并代码啦
别的就都是正常的 git add .  git commit -m "" git push origin master
就可以愉快的随意切换啦~~~~
