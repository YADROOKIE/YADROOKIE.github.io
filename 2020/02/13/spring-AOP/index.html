aop 面向切面编程
将横向重复代码抽离出来 通过代理的方式执行。

实现方式
：1. 接口 + 实现类 由jdk 中的 Proxy 动态代理实现
：2. 实现类 由 cglib字节码增强 实现

术语

  1. target :目标类，需要被代理的类
  2. JoinPoint: 连接点 指那些可能被拦截的方法
  3. PointCut:切入点 已经被增强的连接点
  4. advice: 通知/增强 指增强代码 before after
  5. Weaving:织入 把增强advice 应用到目标对象target 来创建新的代理对象的 过程
  6. Aspect：切面 切入点pointcut 和 通知advice的结合 很多个点互相连接不就是个面了
